## CI папка

Эта папка предназначена для всего, что относится к CI/CD:

- конфигурации пайплайнов (GitLab, GitHub Actions и др.)
- вспомогательные скрипты для деплоя и проверки
- документация по процессу доставки кода

Текущая структура:

- `ci/gitlab` — всё, что относится к GitLab CI/CD
- `ci/github` — будущая конфигурация GitHub Actions (пока только скелет)

Основной файл пайплайна GitLab по требованиям GitLab **остаётся в корне** репозитория (`.gitlab-ci.yml`),  
а в `ci/gitlab` хранятся пояснения и вспомогательные скрипты.
